# Reinforcement Learning Study

基于 *Reinforcement Learning: An Introduction* (Richard S. Sutton & Andrew G. Barto) 的学习项目。

通过编程实现书中的算法和示例，加深对强化学习核心概念的理解。

## 教材

- Sutton & Barto, *Reinforcement Learning: An Introduction* (2nd Edition)
- 在线版本: http://incompleteideas.net/book/the-book.html

## 项目结构

按章节组织代码，逐步添加：

```
ch02/   # Multi-armed Bandits
ch03/   # Finite Markov Decision Processes
ch04/   # Dynamic Programming
ch05/   # Monte Carlo Methods
ch06/   # Temporal-Difference Learning
...
```
